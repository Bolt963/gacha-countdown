Version Countdown

Five game countdowns pinned to Amsterdam time. Use “Check new version” to ask the local backend to search trusted sources, then confirm before updating.

Amsterdam time loading…
How automatic checking works: the button calls your local Node.js backend. The backend searches a small trusted-source list and a web-search fallback, extracts likely version/date/time candidates, and sends them back for review. It does not silently overwrite countdowns because official updates are sometimes posted as images, region-specific notices, or speculative guides.
